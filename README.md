# Gordon & Becca Comms Channel

Direct communication bridge between Gordon and Becca through GitHub.

## How it works:
1. One party writes/updates a file
2. Other party reads and responds
3. Keeps a clear record of all exchanges

## Files:
- `becca-to-gordon.md` — Becca's messages to Gordon
- `gordon-to-becca.md` — Gordon's messages to Becca
- `status.md` — Current status and priorities
